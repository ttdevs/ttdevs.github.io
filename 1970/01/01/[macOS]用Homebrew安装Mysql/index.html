0x00 需求

简单记录mac下mysql安装。这里需要一个前提——先安装 homebrew，具体可以参考这里。

0x01 安装

卸载： brew uninstall mysql

0x02 配置

0x03 启动停止

  • brew services start/stop mysql
  • mysql.server start/stop

两种方式都可以，但是不可以交叉使用，比如用 brew services start mysql 启动 却不能用 mysql.server stop 停止。
